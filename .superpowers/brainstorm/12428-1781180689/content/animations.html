АНИМАЦИИ

Все они уже готовы к реализации — выбирай что берём

GLOW PULSE — ИГРОКИ ЛЕГКО
Свечение игрока дышит раз в ~2 сек. Добавляет жизнь без нагрузки. CSS-анимация поверх canvas-рендера.
SHIMMER — ПЛАТФОРМЫ ЛЕГКО
Блик скользит по платформам слева направо. Делается в canvas render loop — небольшой сдвиг градиента по времени.
SCAN LINE SWEEP ЛЕГКО
Полупрозрачная полоса медленно проходит по канвасу сверху вниз. Усиливает CRT/cyber ощущение. CSS overlay.
PARTICLE TRAIL — ДВИЖЕНИЕ СРЕДНЕ
При движении игрок оставляет 2–4 частицы в цвете команды. Рендерится в canvas, живут ~0.5 сек.
СТАТУС / HUD TRANSITIONS ЛЕГКО
SYNCED
Текст статуса fade + blur при смене (offline → synced). Кнопки пульсируют при hover. CSS transitions.
